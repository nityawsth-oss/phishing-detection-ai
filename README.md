🛡️ SmartShield: AI-Based Real-Time Phishing Detector
An intelligent browser security system that uses Machine Learning to detect phishing websites instantly and protect users while they browse.
🚀 What is this project?
Phishing attacks trick users into revealing sensitive information like passwords and bank details. SmartShield solves this problem by continuously analyzing websites in real time and warning users before any damage is done.
💡 Think of it as your AI-powered security guard for the internet.
🌟 Why this project stands out
⚡ 96.8% Detection Accuracy trained on 235K+ URLs
🌐 Live Website Monitoring through Chrome Extension
🤖 Fully Automated Detection (no manual input needed)
🇮🇳 Optimized for Indian Domains (.gov.in, .edu.in, banking)
🚨 Instant Alerts for suspicious websites
🔥 Core Features
